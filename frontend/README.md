# Frontend (Vite + React + TS)

This folder contains the UI for **FUTURE_FS_03 (ApexFit)**.

See the main project documentation in the repository root: **README.md**.

## Run locally

```bash
npm install
npm run dev
```

The dev server runs on `http://localhost:8080` and proxies API + realtime:

- `/api` → `http://localhost:4000`
- `/ws` → `ws://localhost:4000`
